# DEEPROJECT

Deeproject is a prototype interface for exploring behavioral evaluation of multi-turn AI interactions.

The current version includes:

- a landing page
- an interactive demo view
- a developer-facing evaluation view
- a condensed report view

This prototype is intended for research demonstration and early product exploration.

## Live Demo

Open the GitHub Pages site to view the prototype:

https://stellashiding.github.io/deeproject2026/

Main pages:

- `index.html` — landing page
- `dashboard.html` — interactive demo
- `developer.html` — developer-facing view
- `condensed.html` — condensed report

## Local Usage

Clone the repository and open `index.html` in a browser.

You can also run a local server:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Status

This is a frontend prototype.

The current evaluation behavior is for demonstration purposes only and should not be interpreted as a validated production evaluator.
